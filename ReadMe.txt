Per l'esecuzione del codice seguire i seguenti passaggi:
importare il codice all'interno del proprio IDE 
(consigliato l'utilizzo di PyCharm).

Eseguire l'installazione delle librerie:
numpy
opencv-python
pip
ultralytics==8.0.26
matplotlib

Una volta installati, eseguire nel cmd:
python smokeapp.py
